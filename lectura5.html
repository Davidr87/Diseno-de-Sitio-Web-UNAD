Diferencias entre HTML y HTML5

El estándar HTML se usa y se ha usado desde hace mucho tiempo para la creación de páginas web, casi todas las paginas incluyen dentro de su código algunas etiquetas de este lenguaje de marcas a continuación se mostraran algunas diferencias entre las primeras versiones y el actual HTML5

HTML

* Se escribe en forma de etiquetas rodeadas por corchetes angulares (‹›) y puede describir hasta un cierto punto, la apariencia de un documento.

* Cada atributo y contenido tiene ciertas restricciones para que se considere válida al documento HTML y todas sus variantes abren las instrucciones con ‹› y cierran con ‹›, por ejemplo ‹p›text‹/p›.

* La extensión del archivo en el lenguaje HTML es .html o .htm.

* El lenguaje HTML está diseñado básicamente para desplegar datos y está enfocado a cómo se ven esos datos.

HTML5

* Añade nuevas etiquetas para manejar la Web semántica (Web 3.0).

* Presenta una cantidad de nuevos elementos y algunos atributos que reflejan el uso típico de los sitios web más modernos.

* Introduce nuevas formas de escribir código, así que la sintaxis varía notablemente.

* La extensión del archivo en el lenguaje HTML 5 puede ser .html, .htm. .xhtml, .xht o .xml.
