HTML




Creada aproximadamente en 1980 es el lenguaje utilizado para crear páginas web. Tener claro que no es un lenguaje de programación, es un lenguaje de etiquetas, lo que facilita la escritura, y la interpretación por parte de navegadores como Chrome, IE, Firefox entre otros.
Por ser un lenguaje de etiquetas, las principales son:

DE RAIZ

  • HTML: Es el contenedor de todo el documento .html

DEL DOCUMENTO

  • head: Contiene información sobre el documento, típicamente se alojan datos que los navegadores y rastreadores web
  • title: Se configura el título de la página
  • meta: Describe los metadatos dentro de un documento
  • style: Elemento que permite agregar estilos para el documento. Típicamente se alojan en la etiqueta head, pero funciona también en el body
  • script: Elemento que permite agregar código de programación para el documento. Se pueden llamar scripts externos en archivos formato .js

SECCIONES

  • body: Es el segundo elemento más importante del documento, es el cuerpo del documento
  • nav: Elemento que permite identificar los enlaces de navegación. Por ejemplo, los menús de una aplicación
  • article: Identifica contenido autónomo e independiente del resto del documento
  • aside: Permite el contenido relativamente relacionado con el resto de la página
  • header: Se utiliza para identificar el contenido de la página
  • footer: Sección que permite definir el pie de la página
  • h1, h2, h3, h4, h5, h6: Elementos de cabecera, h1 es la cabecera de mas importancia, h2 es la de menor importancia

CONTENIDO

  • p: Define un párrafo
  • ul: Lista sin orden
  • ol: Lista con orden
  • li: Elementos de una lista
  • div: Es un contenedor genérico. Sirve principalmente para agrupar elementos html

SEMANTICA DE TEXTO

  • a: Elemento que permite crear un enlace
  • br: Permite agregar un salto de línea
  • strong: Establece un texto como importante, es parecido a negrilla
  • b: Agrega negrilla al texto seleccionado
  • i: Agrega cursiva al texto seleccionado
  • u: Agrega subyarado al texto seleccionado
  • span: Elemento para representar texto

CONTENIDO INCRUSTADO

  • img: Etiqueta que se utiliza para agregar una imagen a un documento
  • iframe: Agrega un contenido HTML embebido
  • embed: Agrega un contenido no HTML embebido
  • object: Recurso que permite agregar un elemento externo, como por ejemplo un plugin
  • video: Etiqueta que se utiliza para agregar un video a un documento, agregando la interfaz necesaria para reproducirlo
  • audio: Etiqueta que se utiliza para agregar un sonido a un documento, agregando la interfaz necesaria para reproducirlo

DATOS TABULARES

  • table: Es un elemento que pemite agregar datos tabulares en un documento HTML
  • col: Es una columna de una tabla
  • tr: Es una fila de una tabla
  • td: Es una celda de una tabla (intersección de la fila y la columna
  • th: Es el encabezado de una columna

FORMULARIOS

  • form: Se utiliza para crear un formulario en el document HTML. Se utiliza como contenedor de campos para enviar y/o recibir al servidor
  • label: Tag que representa un campo de texto dentro de un
  • input: Elemento que permite el ingreso de información por parte de un usuario
  • button: Botón que admite como evento predeterminado el clic
  • select: Control que permite visualizar una lista con la posibilidad de seleccionar una opción
  • option: Es un elemento del control anterior
  • textarea: Cuadro de texto multilínea
