Selectores

Son patrones se utilizan para aplicar un mismo estilo a un conjunto de elementos

 

Selector

Ejemplo

Descripción

#id

#PrimerNombre

Únicamente al elemento con nombre PrimerNombre

*

*

Aplica estilo a todos los elementos

element

P

Dar estilo únicamente a los elementos que sea p

element,element

div, p

Hacer que el estilo sea aplicado a div y a p

Element element

div p

Aplicar estilo a los p que se encuentren dentro de un div

:active

a:active

Seleccionar el link activo

::after

p::after

Aplicar estilo después de cada p

:focus

input:focus

Dar estilo cuando el input tenga el foco

:read-only

input:read-only

Hacer que el estilo sea aplicado únicamente cuando los inputs tengan el atributo solo lectura

:link

a:link

Aplica a todos los link no visitados

[attribute=value]

[target=_blank]

Aplicar estilo a todos los elementos que tengan en el atributo target=_blank
